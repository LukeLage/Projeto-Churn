# Projeto Churn Teo me Why

Esse projeto foi feito estudando o curso de Machine Learning do Teo Calvo, do canal Téo Me Why no Youtube.

## Database
Os dados usados para treinar esse modelo podem ser encontrados no link abaixo: 
https://www.kaggle.com/datasets/teocalvo/analytical-base-table-churn/data

## Como rodar o modelo?

Para rodar esse modelo é necessário possuir o Notebook do Jupyter para Python e a biblioteca Anaconda, além das outras instalações citadas abaixo: 


```
pip install pandas
pip install sklearn
pip install feature_engine
```
